Getting started

All packages have been built for Python 3.

Add the packages to your Ubuntu 18.04 by executing the following commands: 

# To add the Bit-Bots signing key:
sudo apt-key adv --keyserver keyserver.ubuntu.com --recv-keys 4C4EDF893374591687621C75C2F8DBB6A37B2874

# To add the package repository:
sudo sh -c 'echo "deb {{ LOCAL_URL }} bionic main" > /etc/apt/sources.list.d/ros.list'

# To remove all previously installed ROS Packages:
sudo apt remove ros-melodic-* && sudo apt update
