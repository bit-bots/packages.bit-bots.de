All packages have been built for Python 3. Add the packages to your Ubuntu 18.04 by executing

  • wget {{ LOCAL_URL }}/key.asc -O- | sudo apt-key add - to add the Bit-Bots signing key
  • sudo sh -c 'echo "deb {{ LOCAL_URL }} bionic main" > /etc/apt/sources.list.d/ros.list' to add the package repository
Remember to remove previous ROS package repositories from your apt lists, to remove all previously installed ROS Packages (sudo apt remove ros-melodic-*) and run sudo apt update.
